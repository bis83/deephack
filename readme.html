THE DEEP HACKER

    「地球は深かった・・・」by Boris Gagarin

地球のコアが爆発しそうなので、大深度地下へとドリル戦車で潜ります。
無敵ドリル戦車はどんな圧力にも耐える装甲と、岩盤を打ち砕くドリルを持っていますが、
完全密閉仕様なので、搭乗者が呼吸できません。

果たして、限られた時間内に地球のコアまでたどり着けるのか！？

はじめ方

deephack.exeを実行してね。
それなりのPCじゃないと動かない(ノートパソコンだと無理かも・・・)ので、
動作しなかったらスミマセン・・・

操作方法

キーボードの↑→↓で方向転換。
スペースキーで加速します。
ゲームパッドを持ってたら、スティックで方向転換、ボタン(何でも)で加速できます。
終わりたくなったらescキーで終了できます。
